####commands####

mkdir -> нужен для создания папки

git add "ваш файл или папка" -> нужен для того чтобы выборочно добавить в коммит файл или папку.

git add . -> нужен для того чтобы добавить файл(-ы) в репозиторий, а . нужна для того чтобы добавить все,
что находится в рамках директории в которой мы находимся

git clone -> для получения и загрузки репозитория к нам на локальный компьютер.

git branch 'ваша ветка' -> нужен для того чтобы создать ветку.

git branch -> для просмотра имеющихся веток и для нахождения ветки в которой находится юзер.

git branch -d "ваша ветка" - нужна для того чтобы удалить ненужную или случайно созданную ветку.

cd .. -> эта команда нужна для того чтобы спускаться на директорию ниже.

git status -> нужен для того, чтобы просматривать состояние текущей ветки(например: есть ли в ней какие либо изменения и тд)

git commit -m "ваши действия" - нужен для того чтобы записывать в историю гита наши действия.

git push origin 'наша ветка' - origin - нужен для того чтобы задать серверу имя по умолчанию, а в целом команда
 нужна для того чтобы вносить изменения или запушить целый проект в удаленный репозиторий(github/gitlab)
